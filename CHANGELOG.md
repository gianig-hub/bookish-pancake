## Version Numbering Scheme

```
MAJOR.MINOR.PATCH

MAJOR — Breaking changes or major feature releases
MINOR — New features, backward-compatible
PATCH — Bug fixes, minor improvements
```

**Pre-release versions:**  
`0.x.x` — Development / MVP phase  
`1.0.0` — Public launch
