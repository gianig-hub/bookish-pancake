# EK Marketplace — User Stories

Core user journeys for the EK Marketplace MVP.

---

## Buyer Stories

### Browse & Search

> As a buyer, I want to search for "used Samsung split AC unit London" so that I can find relevant listings quickly.

> As a buyer, I want to filter listings by category, region, condition, and price so that I see only relevant results.

> As a buyer, I want to see listings sorted by newest, price, and most viewed.

### Viewing Listings

> As a buyer, I want to view a listing detail page with photos, description, price, and seller details.

> As a buyer, I want to contact a seller via a form on the listing page without revealing my email address.

> As a buyer, I want to save listings to a favourites list.

---

## Seller Stories

### Posting Ads

> As a seller, I want to post an equipment listing with up to 10 photos in under 5 minutes.

> As a seller, I want AI assistance to write a better title and description for my listing.

> As a seller, I want to set a category, condition, price, and location for my listing.

### Managing Listings

> As a seller, I want to view all my active listings and edit or deactivate them.

> As a seller, I want to receive email notifications when a buyer contacts me.

> As a seller, I want to mark a listing as sold.

---

## Business / Installer Stories

> As an installer, I want a business profile page showing my services, location, and contact details.

> As an installer, I want to list my services on the marketplace so buyers can find me.

> As an installer, I want to receive leads directly from my listing and profile pages.

---

## Admin Stories

> As an admin, I want to review new listings before they go live (moderation queue).

> As an admin, I want to see an AI-generated spam/scam score for each listing.

> As an admin, I want to deactivate listings, suspend users, and add categories.

> As an admin, I want to view platform metrics (listings, users, messages per day).
