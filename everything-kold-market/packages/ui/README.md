# packages/ui — Shared React Components

Reusable React UI components for EK Marketplace, shared across apps.

## Usage

```tsx
import { Button } from '@ek/ui';
```

## Component Guidelines
- Use TailwindCSS for styling
- Export all components from `src/index.ts`
- Keep components pure and stateless where possible
- Accept `className` prop for overrides
