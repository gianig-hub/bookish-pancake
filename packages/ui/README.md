# packages/ui — EK Marketplace Shared UI Components

Shared React component library for use across all EK Marketplace frontends.

## Usage

```tsx
import { Button, ListingCard } from '@ek/ui'
```

## Component Guidelines

- Use TailwindCSS for styling
- Export all components from `src/index.ts`
- Keep components generic and reusable
- Accept className props for customisation
- Use TypeScript for all props
